## Instalación

Comando de instalación del ejercicio

```bash
git clone https://github.com/Aonik10/coderhouse-practica-integradora-2.git
cd "coderhouse-practica-integradora-2"
npm install
npm run dev
```

El servidor debería iniciarse en el puerto 8080
